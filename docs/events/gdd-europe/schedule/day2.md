# 6 September 2017

## Page Summary

- Day 2 includes a keynote on Google's engineering efforts in Europe and a closing keynote about changing the world one bit at a time.
- There are multiple sessions and Q&A opportunities focused on community building and various Google Developer programs.
- Training sessions cover topics like building data-driven Progressive Web Apps, defining app navigation, using Constraint Layout, Supercharging apps with Cloud Functions and Machine Learning, hands-on TensorFlow for Android, hands-on Google Cloud Vision and Natural Language APIs, hands-on Google Assistant app development, and integrating AMP into PWAs.
- Sessions delve into topics such as Architecture Components, Machine Learning with TensorFlow, fundamentals of Google Cloud Platform, boosting performance through app quality, Panel on Voice UI, Containers, Kubernetes and Google Cloud, Global Tech Policy, Powering apps with location and maps, Frameworks and Tools for Progressive Web Apps, What's next for the Web, Performance tooling, Using Firebase on existing apps, Intro to DeepMind, and Modularizing Android apps.
- The scheduled events are categorized by type (Keynote, Session, Training, Q&A) and track (Android, Beyond Mobile, Community, Develop on Mobile, Mobile Web), with sessions also featuring specific Google products like Assistant, Cloud, TensorFlow, Firebase, and Maps.

All times are Central European Summer Time Zone (UTC+02:00)
