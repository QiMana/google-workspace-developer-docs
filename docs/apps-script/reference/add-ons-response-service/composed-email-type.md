# Enum ComposedEmailType

## Enum ComposedEmailType

An enum value that specifies whether the composed email is a standalone or reply draft.

To call an enum, you call its parent class, name, and property. For example, ` AddOnsResponseService.ComposedEmailType.REPLY_AS_DRAFT`.
