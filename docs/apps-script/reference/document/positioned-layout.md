# Enum PositionedLayout

## Enum PositionedLayout

An enumeration that specifies how to lay out a `PositionedImage` in relation to surrounding text.

To call an enum, you call its parent class, name, and property. For example, ` DocumentApp.PositionedLayout.ABOVE_TEXT`.
