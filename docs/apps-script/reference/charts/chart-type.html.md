# Enum ChartType

## Page Summary

- ChartType represents the chart types supported by the Charts service.
- To call a chart type, you call its parent class, name, and property, for example, `Charts.ChartType.TIMELINE`.
- ChartType includes various chart types such as Timeline, Area, Bar, Bubble, Candlestick, Column, Combo, and more.
