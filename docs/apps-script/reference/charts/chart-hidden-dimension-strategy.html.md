## Enum ChartHiddenDimensionStrategy

An enumeration of how hidden dimensions in a source are expressed in a chart.

To call an enum, you call its parent class, name, and property. For example, ` Charts.ChartHiddenDimensionStrategy.IGNORE_ROWS`.
