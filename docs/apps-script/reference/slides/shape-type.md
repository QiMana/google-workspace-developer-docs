# Enum ShapeType

## Page Summary

- ShapeType is an enum representing various shape types, many of which align with predefined shapes from the ECMA-376 standard.
- You can access an enum value by calling its parent class, name, and property, like `SlidesApp.ShapeType.TEXT_BOX`.
- The ShapeType includes a wide range of shapes such as rectangles, ellipses, various arrows, stars, and flowchart elements.
- There is also an `UNSUPPORTED` shape type and a `CUSTOM` shape type available.

The shape types. Many of these shapes correspond to predefined shapes from the ECMA-376 standard. More information on those shapes can be found in the description of the "ST\_ShapeType" simple type in section 20.1.10.55 of "Office Open XML File Formats - Fundamentals and Markup Language Reference", part 1 of [ECMA-376 5th edition](https://ecma-international.org/publications-and-standards/standards/ecma-376/).

To call an enum, you call its parent class, name, and property. For example, ` SlidesApp.ShapeType.TEXT_BOX`.
