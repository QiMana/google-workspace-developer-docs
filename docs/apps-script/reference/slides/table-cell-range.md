# Class TableCellRange

## Page Summary

- TableCellRange is a collection of one or more TableCell instances.
- The `getTableCells()` method returns a list of TableCell instances.
- Scripts using `getTableCells()` require specific Google API authorization scopes.

#### Authorization

Scripts that use this method require authorization with one or more of the following [scopes](../../concepts/scopes.md#setting_explicit_scopes):

- `https://www.googleapis.com/auth/presentations.currentonly`
- `https://www.googleapis.com/auth/presentations`
