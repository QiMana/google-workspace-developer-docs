# Enum LineType

## Page Summary

- LineType represents various line types derived from a subset of ST\_ShapeType values in ECMA-376.
- To use an enum, you call its parent class, name, and property, such as `SlidesApp.LineType.STRAIGHT_CONNECTOR_1`.
- The available LineType properties include different straight, bent, and curved connectors, as well as a straight line and an unsupported type.

The line types.

Derived from a subset of the values of the "ST\_ShapeType" simple type in section 20.1.10.55 of "Office Open XML File Formats - Fundamentals and Markup Language Reference", part 1 of [ECMA-376 5th edition](https://ecma-international.org/publications-and-standards/standards/ecma-376/).

To call an enum, you call its parent class, name, and property. For example, ` SlidesApp.LineType.STRAIGHT_CONNECTOR_1`.
