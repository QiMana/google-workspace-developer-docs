# Enum CellMergeState

## Page Summary

- CellMergeState defines the possible states of a table cell's merge status.
- You can access enum values using the parent class, name, and property like `SlidesApp.CellMergeState.NORMAL`.
- `NORMAL` indicates a cell is not merged.
- `HEAD` signifies the upper-left cell in a merged set.
- `MERGED` indicates a cell is part of a merged set but is not the head cell.

The table cell merge states.

To call an enum, you call its parent class, name, and property. For example, ` SlidesApp.CellMergeState.NORMAL`.
