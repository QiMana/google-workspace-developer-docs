# Enum PageElementType

## Page Summary

- PageElementType is an enum used to classify different types of page elements.
- You access enum properties by calling its parent class, name, and property.
- The enum includes properties for various page elements like SHAPE, IMAGE, VIDEO, TABLE, and more.
- `UNSUPPORTED` represents a page element that cannot be further classified.

The page element type.

To call an enum, you call its parent class, name, and property. For example, ` SlidesApp.PageElementType.SHAPE`.
