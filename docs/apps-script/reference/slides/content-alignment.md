# Enum ContentAlignment

## Page Summary

- ContentAlignment defines text anchoring types for Shape or TableCell.
- Supported alignments correspond to predefined types from the ECMA-376 standard.
- To use an enum, call its parent class, name, and property, like `SlidesApp.ContentAlignment.TOP`.
- Properties include UNSUPPORTED, TOP, MIDDLE, and BOTTOM alignments.

The content alignments for a `Shape` or `TableCell`. The supported alignments correspond to predefined text anchoring types from the ECMA-376 standard.

To call an enum, you call its parent class, name, and property. For example, ` SlidesApp.ContentAlignment.TOP`.
