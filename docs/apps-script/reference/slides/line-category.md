# Enum LineCategory

## Enum LineCategory

The line category.

The exact `LineType` created is determined based on the category and how it's routed to connect to other page elements.

To call an enum, you call its parent class, name, and property. For example, ` SlidesApp.LineCategory.STRAIGHT`.
