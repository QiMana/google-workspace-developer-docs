# Class CalendarTriggerBuilder

## Page Summary

- CalendarTriggerBuilder is used to build calendar triggers.
- The `create()` method creates and returns the trigger.
- The `onEventUpdated()` method specifies a trigger that fires when a calendar entry is created, updated, or deleted.

Builder for calendar triggers.

## Detailed documentation

### create()

Creates the trigger and returns it.

#### Return

`Trigger` — The new trigger.

---

### onEventUpdated()

Specifies a trigger that fires when a calendar entry is created, updated, or deleted.

#### Return

`CalendarTriggerBuilder` — This `CalendarTriggerBuilder`, for chaining.
