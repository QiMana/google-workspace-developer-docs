# Enum InstallationSource

## Enum InstallationSource

An enumeration that indicates how the script came to be installed as an add-on for the current user.

To call an enum, you call its parent class, name, and property. For example, ` ScriptApp.InstallationSource.APPS_MARKETPLACE_DOMAIN_ADD_ON`.
