# Enum EventType

## Page Summary

- EventType is an enumeration indicating the type of triggered event.
- You access an enum by calling its parent class, name, and property, like `ScriptApp.EventType.CLOCK`.
- The properties listed define different events that can trigger actions, such as time-driven events, opening or editing files, form submissions, changes to files, and calendar event updates.

An enumeration denoting the type of triggered event.

To call an enum, you call its parent class, name, and property. For example, ` ScriptApp.EventType.CLOCK`.
