# Enum Status

## Page Summary

- Status is an enum representing status codes available only for Google Chat apps.
- Enums are called by referencing their parent class, name, and property, like `CardService.Status.OK`.
- The Status enum includes various properties mapping to standard HTTP status codes, such as `OK` (200), `NOT_FOUND` (404), and `INTERNAL` (500).

An enum that represents the status code.

Only available for Google Chat apps. Not available for Google Workspace add-ons.

To call an enum, you call its parent class, name, and property. For example, ` CardService.Status.OK`.
