# Enum ExpressionDataActionType

## Enum ExpressionDataActionType

An enum that represents the type of the [expression data action](https://developers.google.com/apps-script/reference/card-service/expression-data-action).

Only available through the [Gemini Alpha program](https://support.google.com/a/answer/14170809) for Google Workspace add-ons that extend Google Workspace Flows.

```
const expressionDataAction = CardService.newExpressionDataAction()
.setActionType(CardService.ExpressionDataActionType.START_EXPRESSION_EVALUATION);
```
