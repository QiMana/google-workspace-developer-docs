# Class JdbcSQLXML

## Page Summary

- JdbcSQLXML is a JDBC `SQLXML` object with methods for interacting with XML data.
- The `free()` method is used for documentation purposes, referring to `java.sql.SQLXML#free()`.
- The `getString()` method retrieves the string representation of the XML value.
- The `setString(value)` method sets the XML value using a string representation.

A JDBC `SQLXML`. For documentation of this class, see [`java.sql.SQLXML `](https://docs.oracle.com/javase/6/docs/api/java/sql/SQLXML.html)
.
