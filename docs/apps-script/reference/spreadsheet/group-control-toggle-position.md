# Enum GroupControlTogglePosition

## Enum GroupControlTogglePosition

An enumeration representing the possible positions that a group control toggle can have.

To call an enum, you call its parent class, name, and property. For example, ` SpreadsheetApp.GroupControlTogglePosition.BEFORE`.
