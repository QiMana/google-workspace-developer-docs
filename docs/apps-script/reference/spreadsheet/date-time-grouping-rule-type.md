# Enum DateTimeGroupingRuleType

## Page Summary

- DateTimeGroupingRuleType defines types of date-time grouping rules.
- Examples assume the spreadsheet locale is "en-US", and grouping buckets may be translated based on the locale.
- To call an enum, use its parent class, name, and property, like `SpreadsheetApp.DateTimeGroupingRuleType.SECOND`.
- The properties list provides various ways to group date-time data, such as by second, minute, hour, day, month, year, or combinations thereof.

The types of date-time grouping rule.

The examples below assume the spreadsheet `locale` is "en-US", grouping bucket may be translated based on the spreadsheet locale.
