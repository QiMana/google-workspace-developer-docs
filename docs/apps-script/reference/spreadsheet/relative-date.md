# Enum RelativeDate

## Enum RelativeDate

An enumeration representing the relative date options for calculating a value to be used in date-based `BooleanCriteria`.

To call an enum, you call its parent class, name, and property. For example, ` SpreadsheetApp.RelativeDate.TODAY`.
