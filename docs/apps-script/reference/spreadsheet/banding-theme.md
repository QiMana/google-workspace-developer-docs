# Enum BandingTheme

## Page Summary

- BandingTheme is an enumeration that defines various complementary color themes for spreadsheet banding.
- Each theme consists of complementary colors applied to cells based on banding settings.
- You call an enum by using its parent class, name, and property, such as `SpreadsheetApp.BandingTheme.LIGHT_GREY`.
- Several predefined color themes are available as properties of the BandingTheme enum, including LIGHT\_GREY, CYAN, GREEN, and others.

An enumeration of banding themes. Each theme consists of several complementary colors that are applied to different cells based on the banding settings.

To call an enum, you call its parent class, name, and property. For example, ` SpreadsheetApp.BandingTheme.LIGHT_GREY`.
