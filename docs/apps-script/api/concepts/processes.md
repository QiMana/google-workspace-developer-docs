# Processes

Processes are any kind of Google Apps Script function execution that runs on the Apps Script server. Processes can be started through the Apps Script editor, a script trigger, an Google Workspace add-on, a web app, or through the Google Apps Script API itself.

Processes can be listed and examined by means of the Apps Script API [`processes`](https://developers.google.com/apps-script/api/reference/rest/v1/processes) endpoint. The API can provide process information such as script ID, start time, process duration, executing user, status, and other details.
