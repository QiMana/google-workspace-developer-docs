# Chart configuration options

## Page Summary

- This document provides configuration options for various chart types available in Google Apps Script.
- A list of available chart types is presented, including Annotation, Area, Bar, Bubble, Calendar, and many others.
- Detailed configuration options are provided for chart elements such as colors, axes, legends, titles, and chart area dimensions.
- Specific chart types like Annotation and Area charts have dedicated sections outlining their unique configuration properties.

## Chart Types
