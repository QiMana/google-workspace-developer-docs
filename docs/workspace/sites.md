# Access & modify classic Google Sites content programmatically.

**Deprecated**: Sites API is deprecated and might stop working at any time. Sites API can only access [classic Sites](https://gsuiteupdates.googleblog.com/2016/11/a-totally-rebuilt-google-sitesnow.html). Sites API can't access the [rebuilt version of Sites](https://blog.google/products/g-suite/totally-rebuilt-sites-customer-tested/) that was launched on November 22, 2016.

### Entire sites (classic Sites only)

Create new sites or copy existing sites

### Pages, comments & other content (classic Sites only)

Retrieve, create, modify, and delete pages, attachments, and other content.

### Upload and download (classic Sites only)

Upload and download attachments and files.

### Sharing (classic Sites only)

Modify sharing permissions across a site

### Revisions (classic Sites only)

Review the revision history across the site.

### Templates (classic Sites only)

Create page-level templates.
