# Developer guides

The guides in this directory help inform important development decisions.

This directory contains the following guides:

- [Project configuration](https://developers.google.com/workspace/classroom/add-ons/developer-guides/project-configuration) - How to create the required Google Cloud and Google Workspace Marketplace resources behaviors
- [iFrames implementation details](https://developers.google.com/workspace/classroom/add-ons/developer-guides/iframes) - Configuration and technical details for the various add-ons iframes
- [Frictionless sign-in](https://developers.google.com/workspace/classroom/add-ons/developer-guides/frictionless-signin) - Guidance regarding user authentication
- [Interacting with attachments](https://developers.google.com/workspace/classroom/add-ons/developer-guides/attachment-interactions) - How to create attachments and assess student submissions
- [Upgrade links to add-ons](https://developers.google.com/workspace/classroom/add-ons/developer-guides/upgrade-links-to-addons) - How to convert pasted links to add-on attachments
- [Create attachments outside of Classroom](https://developers.google.com/workspace/classroom/add-ons/developer-guides/third-party-first-journey) - How to create attachments from your website or application
- [Prepare for the third-party cookie deprecation](https://developers.google.com/workspace/classroom/add-ons/developer-guides/third-party-cookies) - Recommended courses of action regard upcoming changes to the handling of third-party cookies
- [Turn off or shut down an add-on](https://developers.google.com/workspace/classroom/add-ons/developer-guides/sunset-an-addon) - Options for disabling or removing an add-on application
- [Known issues](https://developers.google.com/workspace/classroom/add-ons/developer-guides/known-issues)
