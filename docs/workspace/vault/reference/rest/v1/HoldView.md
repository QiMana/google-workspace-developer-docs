# HoldView

## HoldView

The amount of detail to return for a hold.

Defaults to <strong>FULL_HOLD</strong>.</td></tr><tr><td><code>BASIC_HOLD</code></td><td>Returns the hold ID, name, update time, service, and query.</td></tr><tr><td><code>FULL_HOLD</code></td><td>Returns all details of <strong>BASIC_HOLD</strong> and the entities the hold applies to, such as accounts or organizational unit.</td></tr></tbody></table>
