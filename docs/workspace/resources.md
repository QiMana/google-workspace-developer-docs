# Google Workspace developer resources

## Page Summary

- The Admin console is used by administrators to manage organizations, including managing third-party app access and other developer tasks.
- The Apps Script dashboard allows you to create projects to automate and extend Google Workspace with code.
- The Google Cloud console is where you can manage the Google Cloud projects that support your Google Workspace integrations.
- The provided resources include tools like the APIs Explorer and Card Builder, along with training and support options such as codelabs and developer support.
- Stay updated on Google Workspace news through the Google Developers blog, the Developer Preview Program, newsletters, Twitter, and YouTube.

In addition to documentation, you can use these tools and resources to build applications, find help, and stay informed about Google Workspace developer products.

## Tools

- [Google Workspace Developer Tools](https://developers.google.com/workspace/guides/developer-tools) — Explore various tools to help you build on Google Workspace.
- [Build with LLMs](https://developers.google.com/workspace/guides/build-with-llms) — Learn how to integrate Large Language Models (LLMs) into your Google Workspace development.
- [Admin console](https://admin.google.com/) — The Google Admin console is where administrators manage their organizations, including allowing third-party apps, enabling domain-wide delegation, and other developer tasks.
- [Apps Script dashboard](https://script.google.com/) — The Apps Script dashboard is where you create Apps Script projects to automate, enhance, or extend Google Workspace with code.
- [Google Cloud console](https://console.cloud.google.com/workspace-api) — The Google Cloud console is where you manage the Google Cloud projects behind your Google Workspace integrations.
- [APIs Explorer](https://developers.google.com/workspace/explore) — Use the Google Workspace APIs Explorer to try out API requests in your browser using real data.
- [Card Builder](https://addons.gsuite.google.com/uikit/builder) — Use the Google Workspace Card Builder to create cards for your solution's UI.

## Training & support

- [How to get started](https://developers.google.com/workspace/guides/get-started) — Follow 5 steps to get started as a Google Workspace developer.
- [Codelabs](https://codelabs.developers.google.com/?product=googleworkspace) — Follow guided, step-by-step instructions to create Apps Script solutions, build a Chat app, or integrate with Google Drive.
- [Developer support](https://developers.google.com/workspace/support) — Review a variety of options to determine the best way to get help with your questions.

## Updates

- [Google Workspace developer release notes](https://developers.google.com/workspace/release-notes) — Learn about recent product changes in Google Workspace.
- [Google Workspace Developers Blog](https://developers.googleblog.com/search/?query=Google+Workspace) — Read the latest news about Google Workspace in the Google Developers blog.
- [Developer Previews](https://developers.google.com/workspace/preview) — Apply to join the Google Workspace Developer Preview Program to get early access to certain features.
- [Newsletter](https://developers.google.com/workspace/newsletters) — Subscribe to the Google Workspace Developer Newsletter to receive a regular emailed summary of new features, upcoming events, and more.
- [Twitter](https://twitter.com/workspacedevs) — Follow *@workspacedevs* on Twitter.
- [YouTube](https://www.youtube.com/@googleworkspacedevs) — Subscribe to the Google Workspace Developers YouTube channel.
