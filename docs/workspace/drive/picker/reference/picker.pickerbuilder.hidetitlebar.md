# Method PickerBuilder.hideTitleBar

## Method PickerBuilder.hideTitleBar

Disable the title bar from being shown. To re-enable, call `setTitle` with a non-empty title or `undefined`.

## Signature

```
hideTitleBar(): PickerBuilder;
```
