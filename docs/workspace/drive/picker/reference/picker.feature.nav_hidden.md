# Enum member Feature.NAV_HIDDEN

## Enum member Feature.NAV\_HIDDEN

Hide the navigation pane. If the navigation pane is hidden, users can only select from the first view chosen.

## Signature

```
NAV_HIDDEN = 'navHidden'
```
