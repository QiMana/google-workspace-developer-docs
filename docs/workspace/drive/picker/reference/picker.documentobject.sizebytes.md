# Property signature DocumentObject.sizeBytes

## Property signature DocumentObject.sizeBytes

Size of the picked item in bytes. The value is not returned when the item is uploaded during the Picker session.

## Signature

```
sizeBytes?: number;
```

## Details

| Type | `number` |
| --- | --- |
| Optional | Yes |
