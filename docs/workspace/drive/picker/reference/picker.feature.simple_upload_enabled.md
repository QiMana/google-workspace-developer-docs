# Enum member Feature.SIMPLE_UPLOAD_ENABLED

## Enum member Feature.SIMPLE\_UPLOAD\_ENABLED

For photo uploads, controls whether per-photo `DocumentObject` (as opposed to per-album) `DocumentObject` is enabled.

## Signature

```
SIMPLE_UPLOAD_ENABLED = 'simpleUploadEnabled'
```
