# Interface ResponseObject

The response object passed to the callback method.

## Signature

```
interface ResponseObject
```

## Property signatures

| Name | Description |
| --- | --- |
| [`action`](./picker.responseobject.action.md) | A type representing the action taken by the user to dismiss the dialog. |
| [`docs`](./picker.responseobject.docs.md) | An array of `DocumentObject` s selected by the user. |
| [`parents`](./picker.responseobject.parents.md) | The parent folders for the selected items. |
| [`viewToken`](./picker.responseobject.viewtoken.md) | The view the user selected these items from. |
