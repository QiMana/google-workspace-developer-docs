# Interface ResponseObject

The response object passed to the callback method.

## Signature

```
interface ResponseObject
```

## Property signatures

| --- | --- |
| [`action`](https://developers.google.com/workspace/drive/picker/reference/picker.responseobject.action) | A type representing the action taken by the user to dismiss the dialog. |
