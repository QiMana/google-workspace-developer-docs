# Display the Google Picker

The Google Picker is a "File Open" dialog for information stored on Google Drive. You can use the [Google Picker API for web apps](../../picker/guides/overview.md) or the [Google Picker API for desktop apps](../../picker/guides/overview-desktop.md) to allow users to open or upload Drive files. The Google Picker API is separate from the Google Drive API.
