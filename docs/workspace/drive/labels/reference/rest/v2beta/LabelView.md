# LabelView

## LabelView

Resource view that can be applied to label responses. The default value `LABEL_VIEW_BASIC` implies the field mask: `name,id,revisionId,labelType,properties.*` \\

<table><colgroup><col width="25%"> <col></colgroup><thead><tr><th colspan="2">Enums</th></tr></thead><tbody><tr><td><code>LABEL_VIEW_BASIC</code></td><td>Implies the field mask: <code>name,id,revisionId,labelType,properties.*</code></td></tr><tr><td><code>LABEL_VIEW_FULL</code></td><td>All possible fields.</td></tr></tbody></table>
