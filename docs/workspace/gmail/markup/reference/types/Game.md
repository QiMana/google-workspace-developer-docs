# Game

Type name: [Game](https://developers.google.com/workspace/gmail/markup/reference/types/Game)

Extends [CreativeWork](https://developers.google.com/workspace/gmail/markup/reference/types/CreativeWork)

| Name | Type | Description |
| --- | --- | --- |
| characterAttribute | [Thing](https://developers.google.com/workspace/gmail/markup/reference/types/Thing) | A piece of data that represents a particular aspect of a fictional character (skill, power, character points, advantage, disadvantage). |
| gameItem | [Thing](https://developers.google.com/workspace/gmail/markup/reference/types/Thing) | An item is an object within the game world that can be collected by a player or, occasionally, a non-player character. |
| gameLocation | [Place](https://developers.google.com/workspace/gmail/markup/reference/types/Place), [PostalAddress](https://developers.google.com/workspace/gmail/markup/reference/types/PostalAddress) or [URL](https://developers.google.com/workspace/gmail/markup/reference/types/URL) | Real or fictional location of the game (or part of game). |
| numberOfPlayers | [QuantitativeValue](https://developers.google.com/workspace/gmail/markup/reference/types/QuantitativeValue) | Indicate how many people can play this game (minimum, maximum, or range). |
| quest | [Thing](https://developers.google.com/workspace/gmail/markup/reference/types/Thing) | The task that a player-controlled character, or group of characters may complete in order to gain a reward. |
