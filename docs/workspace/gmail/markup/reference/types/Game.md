# Game

Type name: [Game](./Game.md)

Extends [CreativeWork](./CreativeWork.md)

| Name | Type | Description |
| --- | --- | --- |
| characterAttribute | [Thing](./Thing.md) | A piece of data that represents a particular aspect of a fictional character (skill, power, character points, advantage, disadvantage). |
| gameItem | [Thing](./Thing.md) | An item is an object within the game world that can be collected by a player or, occasionally, a non-player character. |
| gameLocation | [Place](./Place.md), [PostalAddress](./PostalAddress.md) or [URL](./URL.md) | Real or fictional location of the game (or part of game). |
| numberOfPlayers | [QuantitativeValue](./QuantitativeValue.md) | Indicate how many people can play this game (minimum, maximum, or range). |
| quest | [Thing](./Thing.md) | The task that a player-controlled character, or group of characters may complete in order to gain a reward. |
