# Recipe

Type name: [Recipe](./Recipe.md)

Extends [CreativeWork](./CreativeWork.md)

| Name | Type | Description |
| --- | --- | --- |
| cookingMethod | [Text](./Text.md) | The method of cooking, such as Frying, Steaming,... |
| cookTime | [Duration](./Duration.md) | The time it takes to actually cook the dish, in [ISO 8601 duration format](http://en.wikipedia.org/wiki/ISO_8601). |
| ingredients | [Text](./Text.md) | An ingredient used in the recipe. |
| nutrition | [NutritionInformation](./NutritionInformation.md) | Nutrition information about the recipe. |
| prepTime | [Duration](./Duration.md) | The length of time it takes to prepare the recipe, in [ISO 8601 duration format](http://en.wikipedia.org/wiki/ISO_8601). |
| recipeCategory | [Text](./Text.md) | The category of the recipe—for example, appetizer, entree, etc. |
| recipeCuisine | [Text](./Text.md) | The cuisine of the recipe (for example, French or Ethiopian). |
| recipeInstructions | [Text](./Text.md) | The steps to make the dish. |
| recipeYield | [Text](./Text.md) | The quantity produced by the recipe (for example, number of people served, number of servings, etc). |
| totalTime | [Duration](./Duration.md) | The total time it takes to prepare and cook the recipe, in [ISO 8601 duration format](http://en.wikipedia.org/wiki/ISO_8601). |
