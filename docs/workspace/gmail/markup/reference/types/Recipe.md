# Recipe

Type name: [Recipe](https://developers.google.com/workspace/gmail/markup/reference/types/Recipe)

Extends [CreativeWork](https://developers.google.com/workspace/gmail/markup/reference/types/CreativeWork)

| Name | Type | Description |
| --- | --- | --- |
| cookingMethod | [Text](https://developers.google.com/workspace/gmail/markup/reference/types/Text) | The method of cooking, such as Frying, Steaming,... |
| cookTime | [Duration](https://developers.google.com/workspace/gmail/markup/reference/types/Duration) | The time it takes to actually cook the dish, in [ISO 8601 duration format](http://en.wikipedia.org/wiki/ISO_8601). |
| ingredients | [Text](https://developers.google.com/workspace/gmail/markup/reference/types/Text) | An ingredient used in the recipe. |
| nutrition | [NutritionInformation](https://developers.google.com/workspace/gmail/markup/reference/types/NutritionInformation) | Nutrition information about the recipe. |
| prepTime | [Duration](https://developers.google.com/workspace/gmail/markup/reference/types/Duration) | The length of time it takes to prepare the recipe, in [ISO 8601 duration format](http://en.wikipedia.org/wiki/ISO_8601). |
| recipeCategory | [Text](https://developers.google.com/workspace/gmail/markup/reference/types/Text) | The category of the recipe—for example, appetizer, entree, etc. |
| recipeCuisine | [Text](https://developers.google.com/workspace/gmail/markup/reference/types/Text) | The cuisine of the recipe (for example, French or Ethiopian). |
| recipeInstructions | [Text](https://developers.google.com/workspace/gmail/markup/reference/types/Text) | The steps to make the dish. |
| recipeYield | [Text](https://developers.google.com/workspace/gmail/markup/reference/types/Text) | The quantity produced by the recipe (for example, number of people served, number of servings, etc). |
| totalTime | [Duration](https://developers.google.com/workspace/gmail/markup/reference/types/Duration) | The total time it takes to prepare and cook the recipe, in [ISO 8601 duration format](http://en.wikipedia.org/wiki/ISO_8601). |
