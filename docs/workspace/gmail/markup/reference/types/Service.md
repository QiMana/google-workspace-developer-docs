# Service

Type name: [Service](https://developers.google.com/workspace/gmail/markup/reference/types/Service)

Extends [Intangible](https://developers.google.com/workspace/gmail/markup/reference/types/Intangible)

| Name | Type | Description |
| --- | --- | --- |
| availableChannel | [ServiceChannel](https://developers.google.com/workspace/gmail/markup/reference/types/ServiceChannel) | A means of accessing the service (e.g. a phone bank, a web site, a location, etc.). |
| produces | [Thing](https://developers.google.com/workspace/gmail/markup/reference/types/Thing) | The tangible thing generated by the service, e.g. a passport, permit, etc. |
| provider | [Organization](https://developers.google.com/workspace/gmail/markup/reference/types/Organization) or [Person](https://developers.google.com/workspace/gmail/markup/reference/types/Person) | The organization providing the reservation. |
| serviceArea | [AdministrativeArea](https://developers.google.com/workspace/gmail/markup/reference/types/AdministrativeArea) | The geographic area where the service is provided. |
| serviceAudience | [Audience](https://developers.google.com/workspace/gmail/markup/reference/types/Audience) | The audience eligible for this service. |
| serviceType | [Text](https://developers.google.com/workspace/gmail/markup/reference/types/Text) | The type of service being offered, e.g. veterans' benefits, emergency relief, etc. |
