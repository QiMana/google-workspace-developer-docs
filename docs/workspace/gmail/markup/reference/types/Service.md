# Service

Type name: [Service](./Service.md)

Extends [Intangible](./Intangible.md)

| Name | Type | Description |
| --- | --- | --- |
| availableChannel | [ServiceChannel](./ServiceChannel.md) | A means of accessing the service (e.g. a phone bank, a web site, a location, etc.). |
| produces | [Thing](./Thing.md) | The tangible thing generated by the service, e.g. a passport, permit, etc. |
| provider | [Organization](./Organization.md) or [Person](./Person.md) | The organization providing the reservation. |
| serviceArea | [AdministrativeArea](./AdministrativeArea.md) | The geographic area where the service is provided. |
| serviceAudience | [Audience](./Audience.md) | The audience eligible for this service. |
| serviceType | [Text](./Text.md) | The type of service being offered, e.g. veterans' benefits, emergency relief, etc. |
