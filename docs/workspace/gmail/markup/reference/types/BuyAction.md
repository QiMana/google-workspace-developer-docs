# BuyAction

Type name: [BuyAction](./BuyAction.md)

Extends [TradeAction](./TradeAction.md)

| Name | Type | Description |
| --- | --- | --- |
| seller | [Organization](./Organization.md) or [Person](./Person.md) | An entity which offers (sells / leases / lends / loans) the services / goods. A seller may also be a provider. |
| vendor | [Organization](./Organization.md) or [Person](./Person.md) | 'vendor' is an earlier term for 'seller'. |
| warrantyPromise | [WarrantyPromise](./WarrantyPromise.md) | The warranty promise(s) included in the offer. |
