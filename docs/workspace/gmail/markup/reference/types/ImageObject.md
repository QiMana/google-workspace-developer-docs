# ImageObject

## ImageObject

Type name: [ImageObject](https://developers.google.com/workspace/gmail/markup/reference/types/ImageObject)

Extends [MediaObject](https://developers.google.com/workspace/gmail/markup/reference/types/MediaObject)

| Name | Type | Description |
| --- | --- | --- |
| caption | [Text](https://developers.google.com/workspace/gmail/markup/reference/types/Text) | The caption for this object. |
| exifData | [Text](https://developers.google.com/workspace/gmail/markup/reference/types/Text) | exif data for this object. |
| representativeOfPage | [Boolean](https://developers.google.com/workspace/gmail/markup/reference/types/Boolean) | Indicates whether this image is representative of the content of the page. |
| thumbnail | [ImageObject](https://developers.google.com/workspace/gmail/markup/reference/types/ImageObject) | Thumbnail image for an image or video. |
