# DatedMoneySpecification

Type name: [DatedMoneySpecification](./DatedMoneySpecification.md)

Extends [StructuredValue](./StructuredValue.md)

| Name | Type | Description |
| --- | --- | --- |
| amount | [Number](./Number.md) | The amount of money. |
| currency | [Text](./Text.md) | The currency in which the monetary amount is expressed (in 3-letter [ISO 4217](http://en.wikipedia.org/wiki/ISO_4217) format). |
| endDate |  | The end date and time of the item (in [ISO 8601 date format](http://en.wikipedia.org/wiki/ISO_8601)). |
| startDate |  | The start date and time of the item (in [ISO 8601 date format](http://en.wikipedia.org/wiki/ISO_8601)). |
