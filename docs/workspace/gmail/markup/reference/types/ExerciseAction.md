# ExerciseAction

Type name: [ExerciseAction](./ExerciseAction.md)

Extends [PlayAction](./PlayAction.md)

| Name | Type | Description |
| --- | --- | --- |
| course | [Place](./Place.md) | A sub property of location. The course where this action was taken. |
| diet | [Diet](./Diet.md) | A sub property of instrument. The diet used in this action. |
| distance | [Distance](./Distance.md) | The distance travelled, e.g. exercising or travelling. |
| exercisePlan | [ExercisePlan](./ExercisePlan.md) | A sub property of instrument. The exercise plan used on this action. |
| exerciseType | [Text](./Text.md) | Type(s) of exercise or activity, such as strength training, flexibility training, aerobics, cardiac rehabilitation, etc. |
| fromLocation | [Place](./Place.md) | A sub property of location. The original location of the object or the agent before the action. |
| opponent | [Person](./Person.md) | A sub property of participant. The opponent on this action. |
| sportsActivityLocation | [SportsActivityLocation](./SportsActivityLocation.md) | A sub property of location. The sports activity location where this action occurred. |
| sportsEvent | [SportsEvent](./SportsEvent.md) | A sub property of location. The sports event where this action occurred. |
| sportsTeam | [SportsTeam](./SportsTeam.md) | A sub property of participant. The sports team that participated on this action. |
| toLocation | [Place](./Place.md) | A sub property of location. The final location of the object or the agent after the action. |
