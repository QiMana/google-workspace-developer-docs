# Season

Type name: [Season](./Season.md)

Extends [CreativeWork](./CreativeWork.md)

| Name | Type | Description |
| --- | --- | --- |
| endDate | [Date](./Date.md) | The end date and time of the item (in [ISO 8601 date format](http://en.wikipedia.org/wiki/ISO_8601)). |
| episode | [Episode](./Episode.md) | An episode of a tv, radio or game media within a series or season. |
| episodes | [Episode](./Episode.md) | An episode of a TV/radio series or season. |
| numberOfEpisodes | [Number](./Number.md) | The number of episodes in this season or series. |
| partOfSeries | [Series](./Series.md) | The series to which this episode or season belongs. |
| productionCompany | [Organization](./Organization.md) | The production company or studio responsible for the item e.g. series, video game, episode etc. |
| seasonNumber | [Integer](./Integer.md) or [Text](./Text.md) | Position of the season within an ordered group of seasons. |
| startDate | [Date](./Date.md) | The start date and time of the item (in [ISO 8601 date format](http://en.wikipedia.org/wiki/ISO_8601)). |
| trailer | [VideoObject](./VideoObject.md) | The trailer of a movie or tv/radio series, season, episode, etc. |
