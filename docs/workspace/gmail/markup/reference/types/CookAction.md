# CookAction

Type name: [CookAction](./CookAction.md)

Extends [CreateAction](./CreateAction.md)

| Name | Type | Description |
| --- | --- | --- |
| foodEstablishment | [FoodEstablishment](./FoodEstablishment.md) or [Place](./Place.md) | A sub property of location. The specific food establishment where the action occurred. |
| foodEvent | [FoodEvent](./FoodEvent.md) | A sub property of location. The specific food event where the action occurred. |
| recipe | [Recipe](./Recipe.md) | A sub property of instrument. The recipe/instructions used to perform the action. |
