# MusicPlaylist

Type name: [MusicPlaylist](./MusicPlaylist.md)

Extends [CreativeWork](./CreativeWork.md)

| Name | Type | Description |
| --- | --- | --- |
| numTracks | [Integer](./Integer.md) | The number of tracks in this album or playlist. |
| track | [ItemList](./ItemList.md) or [MusicRecording](./MusicRecording.md) | A music recording (track)—usually a single song. If an ItemList is given, the list should contain items of type MusicRecording. |
| tracks | [MusicRecording](./MusicRecording.md) | A music recording (track)—usually a single song. |
