# Dataset

Type name: [Dataset](./Dataset.md)

Extends [CreativeWork](./CreativeWork.md)

| Name | Type | Description |
| --- | --- | --- |
| catalog | [DataCatalog](./DataCatalog.md) | A data catalog which contains a dataset. |
| distribution | [DataDownload](./DataDownload.md) | A downloadable form of this dataset, at a specific location, in a specific format. |
| spatial | [Place](./Place.md) | The range of spatial applicability of a dataset, e.g. for a dataset of New York weather, the state of New York. |
| temporal | [DateTime](./DateTime.md) | The range of temporal applicability of a dataset, e.g. for a 2011 census dataset, the year 2011 (in ISO 8601 time interval format). |
