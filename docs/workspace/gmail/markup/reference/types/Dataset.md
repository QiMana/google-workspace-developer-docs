# Dataset

Type name: [Dataset](https://developers.google.com/workspace/gmail/markup/reference/types/Dataset)

Extends [CreativeWork](https://developers.google.com/workspace/gmail/markup/reference/types/CreativeWork)

| Name | Type | Description |
| --- | --- | --- |
| catalog | [DataCatalog](https://developers.google.com/workspace/gmail/markup/reference/types/DataCatalog) | A data catalog which contains a dataset. |
| distribution | [DataDownload](https://developers.google.com/workspace/gmail/markup/reference/types/DataDownload) | A downloadable form of this dataset, at a specific location, in a specific format. |
| spatial | [Place](https://developers.google.com/workspace/gmail/markup/reference/types/Place) | The range of spatial applicability of a dataset, e.g. for a dataset of New York weather, the state of New York. |
| temporal | [DateTime](https://developers.google.com/workspace/gmail/markup/reference/types/DateTime) | The range of temporal applicability of a dataset, e.g. for a 2011 census dataset, the year 2011 (in ISO 8601 time interval format). |
