# CommunicateAction

Type name: [CommunicateAction](./CommunicateAction.md)

Extends [InteractAction](./InteractAction.md)

| Name | Type | Description |
| --- | --- | --- |
| about | [Thing](./Thing.md) | The subject matter of the content. |
| language | [Language](./Language.md) | A sub property of instrument. The language used on this action. |
| recipient | [Audience](./Audience.md), [Organization](./Organization.md) or [Person](./Person.md) | A sub property of participant. The participant who is at the receiving end of the action. |
