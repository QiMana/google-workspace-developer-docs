# ExercisePlan

Type name: [ExercisePlan](./ExercisePlan.md)

Extends [CreativeWork](./CreativeWork.md) or [PhysicalActivity](./PhysicalActivity.md)

| Name | Type | Description |
| --- | --- | --- |
| activityDuration | [Duration](./Duration.md) | Length of time to engage in the activity. |
| activityFrequency | [Text](./Text.md) | How often one should engage in the activity. |
| additionalVariable | [Text](./Text.md) | Any additional component of the exercise prescription that may need to be articulated to the patient. This may include the order of exercises, the number of repetitions of movement, quantitative distance, progressions over time, etc. |
| exerciseType | [Text](./Text.md) | Type(s) of exercise or activity, such as strength training, flexibility training, aerobics, cardiac rehabilitation, etc. |
| intensity | [Text](./Text.md) | Quantitative measure gauging the degree of force involved in the exercise, for example, heartbeats per minute. May include the velocity of the movement. |
| repetitions | [Number](./Number.md) | Number of times one should repeat the activity. |
| restPeriods | [Text](./Text.md) | How often one should break from the activity. |
| workload | [Energy](./Energy.md) | Quantitative measure of the physiologic output of the exercise; also referred to as energy expenditure. |
