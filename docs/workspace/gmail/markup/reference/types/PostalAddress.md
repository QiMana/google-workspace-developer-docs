# PostalAddress

Type name: [PostalAddress](./PostalAddress.md)

Extends [ContactPoint](./ContactPoint.md)

| Name | Type | Description |
| --- | --- | --- |
| addressCountry | [Country](./Country.md) | The country. For example, USA. You can also provide the two-letter [ISO 3166-1 alpha-2 country code](http://en.wikipedia.org/wiki/ISO_3166-1). |
| addressLocality | [Text](./Text.md) | The locality. For example, Mountain View. |
| addressRegion | [Text](./Text.md) | The region. For example, CA. |
| postalCode | [Text](./Text.md) | The postal code. For example, 94043. |
| postOfficeBoxNumber | [Text](./Text.md) | The post office box number for PO box addresses. |
| streetAddress | [Text](./Text.md) | The street address. For example, 1600 Amphitheatre Pkwy. |
