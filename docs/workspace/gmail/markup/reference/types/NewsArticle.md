# NewsArticle

Type name: [NewsArticle](https://developers.google.com/workspace/gmail/markup/reference/types/NewsArticle)

Extends [Article](https://developers.google.com/workspace/gmail/markup/reference/types/Article)

| Name | Type | Description |
| --- | --- | --- |
| dateline | [Text](https://developers.google.com/workspace/gmail/markup/reference/types/Text) | The location where the NewsArticle was produced. |
| printColumn | [Text](https://developers.google.com/workspace/gmail/markup/reference/types/Text) | The number of the column in which the NewsArticle appears in the print edition. |
| printEdition | [Text](https://developers.google.com/workspace/gmail/markup/reference/types/Text) | The edition of the print product in which the NewsArticle appears. |
| printPage | [Text](https://developers.google.com/workspace/gmail/markup/reference/types/Text) | If this NewsArticle appears in print, this field indicates the name of the page on which the article is found. Please note that this field is intended for the exact page name (e.g. A5, B18). |
| printSection | [Text](https://developers.google.com/workspace/gmail/markup/reference/types/Text) | If this NewsArticle appears in print, this field indicates the print section in which the article appeared. |
