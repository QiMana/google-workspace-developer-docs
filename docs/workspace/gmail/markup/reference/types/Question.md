# Question

Type name: [Question](./Question.md)

Extends [CreativeWork](./CreativeWork.md)

| Name | Type | Description |
| --- | --- | --- |
| acceptedAnswer | [Answer](./Answer.md) | The answer that has been accepted as best, typically on a Question/Answer site. Sites vary in their selection mechanisms, e.g. drawing on community opinion and/or the view of the Question author. |
| answerCount | [Integer](./Integer.md) | The number of answers this question has received. |
| downvoteCount | [Integer](./Integer.md) | The number of downvotes this question has received from the community. |
| suggestedAnswer | [Answer](./Answer.md) | An answer (possibly one of several, possibly incorrect) to a Question, e.g. on a Question/Answer site. |
| upvoteCount | [Integer](./Integer.md) | The number of upvotes this question has received from the community. |
