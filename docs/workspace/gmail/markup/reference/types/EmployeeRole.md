# EmployeeRole

## EmployeeRole

Type name: [EmployeeRole](./EmployeeRole.md)

Extends [OrganizationRole](./OrganizationRole.md)

| Name | Type | Description |
| --- | --- | --- |
| baseSalary | [Number](./Number.md) or [PriceSpecification](./PriceSpecification.md) | The base salary of the job or of an employee in an EmployeeRole. |
| salaryCurrency | [Text](./Text.md) | The currency (coded using ISO 4217, http://en.wikipedia.org/wiki/ISO\_4217 ) used for the main salary information in this job posting or for this employee. |
