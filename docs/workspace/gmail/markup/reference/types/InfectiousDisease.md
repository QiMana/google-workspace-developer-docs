# InfectiousDisease

Type name: [InfectiousDisease](./InfectiousDisease.md)

Extends [MedicalCondition](./MedicalCondition.md)

| Name | Type | Description |
| --- | --- | --- |
| infectiousAgent | [Text](./Text.md) | The actual infectious agent, such as a specific bacterium. |
| infectiousAgentClass | [InfectiousAgentClass](./InfectiousAgentClass.md) | The class of infectious agent (bacteria, prion, etc.) that causes the disease. |
| transmissionMethod | [Text](./Text.md) | How the disease spreads, either as a route or vector, for example 'direct contact', 'Aedes aegypti', etc. |
