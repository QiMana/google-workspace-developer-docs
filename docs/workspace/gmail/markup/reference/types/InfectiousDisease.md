Type name: [InfectiousDisease](https://developers.google.com/workspace/gmail/markup/reference/types/InfectiousDisease)

| Name | Type | Description |
| --- | --- | --- |
| infectiousAgent | [Text](https://developers.google.com/workspace/gmail/markup/reference/types/Text) | The actual infectious agent, such as a specific bacterium. |
