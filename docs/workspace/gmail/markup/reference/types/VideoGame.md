# VideoGame

Type name: [VideoGame](./VideoGame.md)

Extends [Game](./Game.md) or [SoftwareApplication](./SoftwareApplication.md)

| Name | Type | Description |
| --- | --- | --- |
| actor | [Person](./Person.md) | An actor, e.g. in tv, radio, movie, video games etc. Actors can be associated with individual items or with a series, episode, clip. |
| actors | [Person](./Person.md) | An actor, e.g. in tv, radio, movie, video games etc. Actors can be associated with individual items or with a series, episode, clip. |
| cheatCode | [CreativeWork](./CreativeWork.md) | Cheat codes to the game. |
| director | [Person](./Person.md) | A director of e.g. tv, radio, movie, video games etc. content. Directors can be associated with individual items or with a series, episode, clip. |
| directors | [Person](./Person.md) | A director of e.g. tv, radio, movie, video games etc. content. Directors can be associated with individual items or with a series, episode, clip. |
| gamePlatform | [Text](./Text.md), [Thing](./Thing.md) or [URL](./URL.md) | The electronic systems used to play [video games](http://en.wikipedia.org/wiki/Category:Video_game_platforms). |
| gameServer | [GameServer](./GameServer.md) | The server on which it is possible to play the game. |
| gameTip | [CreativeWork](./CreativeWork.md) | Links to tips, tactics, etc. |
| musicBy | [MusicGroup](./MusicGroup.md) or [Person](./Person.md) | The composer of the soundtrack. |
| playMode | [GamePlayMode](./GamePlayMode.md) | Indicates whether this game is multi-player, co-op or single-player. The game can be marked as multi-player, co-op and single-player at the same time. |
| trailer | [VideoObject](./VideoObject.md) | The trailer of a movie or tv/radio series, season, episode, etc. |
