# AlignmentObject

Type name: [AlignmentObject](./AlignmentObject.md)

Extends [Intangible](./Intangible.md)

| Name | Type | Description |
| --- | --- | --- |
| alignmentType | [Text](./Text.md) | A category of alignment between the learning resource and the framework node. Recommended values include: 'assesses', 'teaches', 'requires', 'textComplexity', 'readingLevel', 'educationalSubject', and 'educationLevel'. |
| educationalFramework | [Text](./Text.md) | The framework to which the resource being described is aligned. |
| targetDescription | [Text](./Text.md) | The description of a node in an established educational framework. |
| targetName | [Text](./Text.md) | The name of a node in an established educational framework. |
| targetUrl | [URL](./URL.md) | The URL of a node in an established educational framework. |
