# PublicationVolume

Type name: [PublicationVolume](https://developers.google.com/workspace/gmail/markup/reference/types/PublicationVolume)

Extends [CreativeWork](https://developers.google.com/workspace/gmail/markup/reference/types/CreativeWork)

| Name | Type | Description |
| --- | --- | --- |
| pageEnd | [Integer](https://developers.google.com/workspace/gmail/markup/reference/types/Integer) or [Text](https://developers.google.com/workspace/gmail/markup/reference/types/Text) | The page on which the work ends; for example "138" or "xvi". |
| pageStart | [Integer](https://developers.google.com/workspace/gmail/markup/reference/types/Integer) or [Text](https://developers.google.com/workspace/gmail/markup/reference/types/Text) | The page on which the work starts; for example "135" or "xiii". |
| pagination | [Text](https://developers.google.com/workspace/gmail/markup/reference/types/Text) | Any description of pages that is not separated into pageStart and pageEnd; for example, "1-6, 9, 55" or "10-12, 46-49". |
| volumeNumber | [Integer](https://developers.google.com/workspace/gmail/markup/reference/types/Integer) or [Text](https://developers.google.com/workspace/gmail/markup/reference/types/Text) | Identifies the volume of publication or multi-part work; for example, "iii" or "2". |
