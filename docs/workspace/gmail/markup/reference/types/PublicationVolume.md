# PublicationVolume

Type name: [PublicationVolume](./PublicationVolume.md)

Extends [CreativeWork](./CreativeWork.md)

| Name | Type | Description |
| --- | --- | --- |
| pageEnd | [Integer](./Integer.md) or [Text](./Text.md) | The page on which the work ends; for example "138" or "xvi". |
| pageStart | [Integer](./Integer.md) or [Text](./Text.md) | The page on which the work starts; for example "135" or "xiii". |
| pagination | [Text](./Text.md) | Any description of pages that is not separated into pageStart and pageEnd; for example, "1-6, 9, 55" or "10-12, 46-49". |
| volumeNumber | [Integer](./Integer.md) or [Text](./Text.md) | Identifies the volume of publication or multi-part work; for example, "iii" or "2". |
