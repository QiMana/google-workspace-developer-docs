# ListItem

Type name: [ListItem](https://developers.google.com/workspace/gmail/markup/reference/types/ListItem)

Extends [Intangible](https://developers.google.com/workspace/gmail/markup/reference/types/Intangible)

| Name | Type | Description |
| --- | --- | --- |
| item | [Thing](https://developers.google.com/workspace/gmail/markup/reference/types/Thing) | An entity represented by an entry in a list (e.g. an 'artist' in a list of 'artists')’. |
| nextItem | [ListItem](https://developers.google.com/workspace/gmail/markup/reference/types/ListItem) | A link to the ListItem that follows the current one. |
| position | [Integer](https://developers.google.com/workspace/gmail/markup/reference/types/Integer) or [Text](https://developers.google.com/workspace/gmail/markup/reference/types/Text) | The position of an item in a series or sequence of items. |
| previousItem | [ListItem](https://developers.google.com/workspace/gmail/markup/reference/types/ListItem) | A link to the ListItem that precedes the current one. |
