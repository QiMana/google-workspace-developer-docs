# ListItem

Type name: [ListItem](./ListItem.md)

Extends [Intangible](./Intangible.md)

| Name | Type | Description |
| --- | --- | --- |
| item | [Thing](./Thing.md) | An entity represented by an entry in a list (e.g. an 'artist' in a list of 'artists')’. |
| nextItem | [ListItem](./ListItem.md) | A link to the ListItem that follows the current one. |
| position | [Integer](./Integer.md) or [Text](./Text.md) | The position of an item in a series or sequence of items. |
| previousItem | [ListItem](./ListItem.md) | A link to the ListItem that precedes the current one. |
