# TrainTrip

Type name: [TrainTrip](https://developers.google.com/workspace/gmail/markup/reference/types/TrainTrip)

Extends [Intangible](https://developers.google.com/workspace/gmail/markup/reference/types/Intangible)

| Name | Type | Description |
| --- | --- | --- |
| arrivalPlatform | [Text](https://developers.google.com/workspace/gmail/markup/reference/types/Text) | The platform where the train arrives. |
| arrivalStation | [TrainStation](https://developers.google.com/workspace/gmail/markup/reference/types/TrainStation) | The station where the train trip ends. |
| arrivalTime | [DateTime](https://developers.google.com/workspace/gmail/markup/reference/types/DateTime) | The expected arrival time. |
| departurePlatform | [Text](https://developers.google.com/workspace/gmail/markup/reference/types/Text) | The platform from which the train departs. |
| departureStation | [TrainStation](https://developers.google.com/workspace/gmail/markup/reference/types/TrainStation) | The station from which the train departs. |
| departureTime | [DateTime](https://developers.google.com/workspace/gmail/markup/reference/types/DateTime) | The expected departure time. |
| provider | [Organization](https://developers.google.com/workspace/gmail/markup/reference/types/Organization) or [Person](https://developers.google.com/workspace/gmail/markup/reference/types/Person) | The organization providing the reservation. |
| trainName | [Text](https://developers.google.com/workspace/gmail/markup/reference/types/Text) | The name of the train (e.g. The Orient Express). |
| trainNumber | [Text](https://developers.google.com/workspace/gmail/markup/reference/types/Text) | The unique identifier for the train. |
