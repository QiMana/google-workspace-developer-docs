# VisualArtwork

Type name: [VisualArtwork](https://developers.google.com/workspace/gmail/markup/reference/types/VisualArtwork)

Extends [CreativeWork](https://developers.google.com/workspace/gmail/markup/reference/types/CreativeWork)

| Name | Type | Description |
| --- | --- | --- |
| artEdition | [Integer](https://developers.google.com/workspace/gmail/markup/reference/types/Integer) or [Text](https://developers.google.com/workspace/gmail/markup/reference/types/Text) | The number of copies when multiple copies of a piece of artwork are produced - e.g. for a limited edition of 20 prints, 'artEdition' refers to the total number of copies (in this example "20"). |
| artform | [Text](https://developers.google.com/workspace/gmail/markup/reference/types/Text) or [URL](https://developers.google.com/workspace/gmail/markup/reference/types/URL) | e.g. Painting, Drawing, Sculpture, Print, Photograph, Assemblage, Collage, etc. |
| depth | [Distance](https://developers.google.com/workspace/gmail/markup/reference/types/Distance) or [QuantitativeValue](https://developers.google.com/workspace/gmail/markup/reference/types/QuantitativeValue) | The depth of the item. |
| height | [Distance](https://developers.google.com/workspace/gmail/markup/reference/types/Distance) or [QuantitativeValue](https://developers.google.com/workspace/gmail/markup/reference/types/QuantitativeValue) | The height of the item. |
| material | [Text](https://developers.google.com/workspace/gmail/markup/reference/types/Text) or [URL](https://developers.google.com/workspace/gmail/markup/reference/types/URL) | e.g. Oil, Watercolour, Acrylic, Linoprint, Marble, Cyanotype, Digital, Lithograph, DryPoint, Intaglio, Pastel, Woodcut, Pencil, Mixed Media, etc. |
| surface | [Text](https://developers.google.com/workspace/gmail/markup/reference/types/Text) or [URL](https://developers.google.com/workspace/gmail/markup/reference/types/URL) | e.g. Canvas, Paper, Wood, Board, etc. |
| width | [Distance](https://developers.google.com/workspace/gmail/markup/reference/types/Distance) or [QuantitativeValue](https://developers.google.com/workspace/gmail/markup/reference/types/QuantitativeValue) | The width of the item. |
