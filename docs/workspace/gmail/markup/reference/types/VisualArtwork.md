# VisualArtwork

Type name: [VisualArtwork](./VisualArtwork.md)

Extends [CreativeWork](./CreativeWork.md)

| Name | Type | Description |
| --- | --- | --- |
| artEdition | [Integer](./Integer.md) or [Text](./Text.md) | The number of copies when multiple copies of a piece of artwork are produced - e.g. for a limited edition of 20 prints, 'artEdition' refers to the total number of copies (in this example "20"). |
| artform | [Text](./Text.md) or [URL](./URL.md) | e.g. Painting, Drawing, Sculpture, Print, Photograph, Assemblage, Collage, etc. |
| depth | [Distance](./Distance.md) or [QuantitativeValue](./QuantitativeValue.md) | The depth of the item. |
| height | [Distance](./Distance.md) or [QuantitativeValue](./QuantitativeValue.md) | The height of the item. |
| material | [Text](./Text.md) or [URL](./URL.md) | e.g. Oil, Watercolour, Acrylic, Linoprint, Marble, Cyanotype, Digital, Lithograph, DryPoint, Intaglio, Pastel, Woodcut, Pencil, Mixed Media, etc. |
| surface | [Text](./Text.md) or [URL](./URL.md) | e.g. Canvas, Paper, Wood, Board, etc. |
| width | [Distance](./Distance.md) or [QuantitativeValue](./QuantitativeValue.md) | The width of the item. |
