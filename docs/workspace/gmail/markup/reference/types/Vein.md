# Vein

Type name: [Vein](./Vein.md)

Extends [Vessel](./Vessel.md)

| Name | Type | Description |
| --- | --- | --- |
| drainsTo | [Vessel](./Vessel.md) | The vasculature that the vein drains into. |
| regionDrained | [AnatomicalStructure](./AnatomicalStructure.md) or [AnatomicalSystem](./AnatomicalSystem.md) | The anatomical or organ system drained by this vessel; generally refers to a specific part of an organ. |
| tributary | [AnatomicalStructure](./AnatomicalStructure.md) | The anatomical or organ system that the vein flows into; a larger structure that the vein connects to. |
