# MusicGroup

Type name: [MusicGroup](https://developers.google.com/workspace/gmail/markup/reference/types/MusicGroup)

Extends [PerformingGroup](https://developers.google.com/workspace/gmail/markup/reference/types/PerformingGroup)

| Name | Type | Description |
| --- | --- | --- |
| album | [MusicAlbum](https://developers.google.com/workspace/gmail/markup/reference/types/MusicAlbum) | A music album. |
| albums | [MusicAlbum](https://developers.google.com/workspace/gmail/markup/reference/types/MusicAlbum) | A collection of music albums. |
| genre | [Text](https://developers.google.com/workspace/gmail/markup/reference/types/Text) | Genre of the creative work or group. |
| musicGroupMember | [Person](https://developers.google.com/workspace/gmail/markup/reference/types/Person) | A member of a music group—for example, John, Paul, George, or Ringo. |
| track | [ItemList](https://developers.google.com/workspace/gmail/markup/reference/types/ItemList) or [MusicRecording](https://developers.google.com/workspace/gmail/markup/reference/types/MusicRecording) | A music recording (track)—usually a single song. If an ItemList is given, the list should contain items of type MusicRecording. |
| tracks | [MusicRecording](https://developers.google.com/workspace/gmail/markup/reference/types/MusicRecording) | A music recording (track)—usually a single song. |
