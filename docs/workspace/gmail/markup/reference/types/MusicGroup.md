# MusicGroup

Type name: [MusicGroup](./MusicGroup.md)

Extends [PerformingGroup](./PerformingGroup.md)

| Name | Type | Description |
| --- | --- | --- |
| album | [MusicAlbum](./MusicAlbum.md) | A music album. |
| albums | [MusicAlbum](./MusicAlbum.md) | A collection of music albums. |
| genre | [Text](./Text.md) | Genre of the creative work or group. |
| musicGroupMember | [Person](./Person.md) | A member of a music group—for example, John, Paul, George, or Ringo. |
| track | [ItemList](./ItemList.md) or [MusicRecording](./MusicRecording.md) | A music recording (track)—usually a single song. If an ItemList is given, the list should contain items of type MusicRecording. |
| tracks | [MusicRecording](./MusicRecording.md) | A music recording (track)—usually a single song. |
