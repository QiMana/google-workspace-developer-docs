# Rating

Type name: [Rating](./Rating.md)

Extends [Intangible](./Intangible.md)

| Name | Type | Description |
| --- | --- | --- |
| bestRating | [Number](./Number.md) or [Text](./Text.md) | The highest value allowed in this rating system. If bestRating is omitted, 5 is assumed. |
| ratingValue | [Text](./Text.md) | The rating for the content. |
| worstRating | [Number](./Number.md) or [Text](./Text.md) | The lowest value allowed in this rating system. If worstRating is omitted, 1 is assumed. |
