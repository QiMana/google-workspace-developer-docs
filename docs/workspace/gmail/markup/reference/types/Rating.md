# Rating

Type name: [Rating](https://developers.google.com/workspace/gmail/markup/reference/types/Rating)

Extends [Intangible](https://developers.google.com/workspace/gmail/markup/reference/types/Intangible)

| Name | Type | Description |
| --- | --- | --- |
| bestRating | [Number](https://developers.google.com/workspace/gmail/markup/reference/types/Number) or [Text](https://developers.google.com/workspace/gmail/markup/reference/types/Text) | The highest value allowed in this rating system. If bestRating is omitted, 5 is assumed. |
| ratingValue | [Text](https://developers.google.com/workspace/gmail/markup/reference/types/Text) | The rating for the content. |
| worstRating | [Number](https://developers.google.com/workspace/gmail/markup/reference/types/Number) or [Text](https://developers.google.com/workspace/gmail/markup/reference/types/Text) | The lowest value allowed in this rating system. If worstRating is omitted, 1 is assumed. |
