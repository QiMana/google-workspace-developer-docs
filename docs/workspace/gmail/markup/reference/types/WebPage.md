# WebPage

Type name: [WebPage](https://developers.google.com/workspace/gmail/markup/reference/types/WebPage)

Extends [CreativeWork](https://developers.google.com/workspace/gmail/markup/reference/types/CreativeWork)

| Name | Type | Description |
| --- | --- | --- |
| breadcrumb | [BreadcrumbList](https://developers.google.com/workspace/gmail/markup/reference/types/BreadcrumbList) or [Text](https://developers.google.com/workspace/gmail/markup/reference/types/Text) | A set of links that can help a user understand and navigate a website hierarchy. |
| lastReviewed | [Date](https://developers.google.com/workspace/gmail/markup/reference/types/Date) | Date on which the content on this web page was last reviewed for accuracy and/or completeness. |
| mainContentOfPage | [WebPageElement](https://developers.google.com/workspace/gmail/markup/reference/types/WebPageElement) | Indicates if this web page element is the main subject of the page. |
| primaryImageOfPage | [ImageObject](https://developers.google.com/workspace/gmail/markup/reference/types/ImageObject) | Indicates the main image on the page. |
| relatedLink | [URL](https://developers.google.com/workspace/gmail/markup/reference/types/URL) | A link related to this web page, for example to other related web pages. |
| reviewedBy | [Organization](https://developers.google.com/workspace/gmail/markup/reference/types/Organization) or [Person](https://developers.google.com/workspace/gmail/markup/reference/types/Person) | People or organizations that have reviewed the content on this web page for accuracy and/or completeness. |
| significantLink | [URL](https://developers.google.com/workspace/gmail/markup/reference/types/URL) | One of the more significant URLs on the page. Typically, these are the non-navigation links that are clicked on the most. |
| significantLinks | [URL](https://developers.google.com/workspace/gmail/markup/reference/types/URL) | The most significant URLs on the page. Typically, these are the non-navigation links that are clicked on the most. |
| specialty | [Specialty](https://developers.google.com/workspace/gmail/markup/reference/types/Specialty) | One of the domain specialities to which this web page's content applies. |
