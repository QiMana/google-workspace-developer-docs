# WebPage

Type name: [WebPage](./WebPage.md)

Extends [CreativeWork](./CreativeWork.md)

| Name | Type | Description |
| --- | --- | --- |
| breadcrumb | [BreadcrumbList](./BreadcrumbList.md) or [Text](./Text.md) | A set of links that can help a user understand and navigate a website hierarchy. |
| lastReviewed | [Date](./Date.md) | Date on which the content on this web page was last reviewed for accuracy and/or completeness. |
| mainContentOfPage | [WebPageElement](./WebPageElement.md) | Indicates if this web page element is the main subject of the page. |
| primaryImageOfPage | [ImageObject](./ImageObject.md) | Indicates the main image on the page. |
| relatedLink | [URL](./URL.md) | A link related to this web page, for example to other related web pages. |
| reviewedBy | [Organization](./Organization.md) or [Person](./Person.md) | People or organizations that have reviewed the content on this web page for accuracy and/or completeness. |
| significantLink | [URL](./URL.md) | One of the more significant URLs on the page. Typically, these are the non-navigation links that are clicked on the most. |
| significantLinks | [URL](./URL.md) | The most significant URLs on the page. Typically, these are the non-navigation links that are clicked on the most. |
| specialty | [Specialty](./Specialty.md) | One of the domain specialities to which this web page's content applies. |
