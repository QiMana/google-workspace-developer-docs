# Corporation

Type name: [Corporation](./Corporation.md)

Extends [Organization](./Organization.md)

| Name | Type | Description |
| --- | --- | --- |
| tickerSymbol | [Text](./Text.md) | The exchange traded instrument associated with a Corporation object. The tickerSymbol is expressed as an exchange and an instrument name separated by a space character. For the exchange component of the tickerSymbol attribute, we recommend using the controlled vocaulary of Market Identifier Codes (MIC) specified in ISO15022. |
