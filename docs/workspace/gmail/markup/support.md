## Ask a Question about Schemas

You may ask questions about Gmail schemas supported on [Stack Overflow](http://stackoverflow.com/questions/ask?tags=google-schemas). Google engineers monitor and answer against the tag [google-schemas](http://stackoverflow.com/questions/tagged/google-schemas), so please use this tag when asking questions. We aim to answer all questions in a reasonable period of time.
