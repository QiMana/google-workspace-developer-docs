# Supported platforms

## Desktop browser compatibility table

Gmail's AMP rendering relies on modern browser features which are not supported by all web browsers.

On incompatible browsers, emails fallback to HTML.

| Browser | Minimum version |
| --- | --- |
| Chrome | 69 |
| Edge | *Not supported* |
| Firefox | 58 |
| Internet Explorer | *Not supported* |
| Opera | 48 |
| Safari | 10 |

## Mobile support

Dynamic emails are supported in the latest version of the [official Gmail app for iOS](https://apps.apple.com/us/app/gmail-email-by-google/id422689480) and [official Gmail app for Android](https://play.google.com/store/apps/details?id=com.google.android.gm).

On Android, the minimum required OS version is 5.0 (Android Lollipop) with the system WebView version of 74 or above.
