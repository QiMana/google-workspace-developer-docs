# Preview your annotations

To preview your email annotations, edit the following code or write new code. To find JSON-LD and Microdata templates, see [Get started](./overview.md).

This Preview Tool provides a glimpse of how your annotation will appear in your subscribers' Promotions Tab. It also detects potential issues with your annotation, offering error messages to help you address them. The tool checks for compliance with image quality filters, proper syntax, supported image aspect ratios and sizes, valid image formats, and supported URLs.
