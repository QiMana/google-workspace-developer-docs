# Google Cloud Search reference

Welcome to the Google Cloud reference documentation. This documentation contains the following sections:
- [Google-supplied connector parameters](https://developers.google.com/workspace/cloud-search/docs/reference/connector-configuration) - Description of all pre-defined connector parameters provided by Google.
- [Google Cloud Search API](https://developers.google.com/workspace/cloud-search/docs/reference/rest) - Used to index non-Google Workspace data for Google Cloud Search. The REST API is best suited for developers who are not programming in Java.
- [Search Widget](https://developers.google.com/workspace/cloud-search/docs/reference/widget/js) - Used to implement the search widget as your web search interface.
- [Cloud Search Java SDK](https://developers.google.com/workspace/cloud-search/docs/reference/sdk) - Used to index non-Google Workspace data for Google Cloud Search. The SDK is best suited for developers who are programming in Java.
- [Schemas](https://developers.google.com/workspace/cloud-search/docs/reference/schemas) - Well-known schemas and reserved schema operators.
