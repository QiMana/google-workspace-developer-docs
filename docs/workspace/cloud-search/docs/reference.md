# Google Cloud Search reference

Welcome to the Google Cloud reference documentation. This documentation contains the following sections:
- [Google-supplied connector parameters](./reference/connector-configuration.md) - Description of all pre-defined connector parameters provided by Google.
- [Google Cloud Search API](./reference/rest.md) - Used to index non-Google Workspace data for Google Cloud Search. The REST API is best suited for developers who are not programming in Java.
- [Search Widget](./reference/widget/js.md) - Used to implement the search widget as your web search interface.
- [Cloud Search Java SDK](./reference/sdk.md) - Used to index non-Google Workspace data for Google Cloud Search. The SDK is best suited for developers who are programming in Java.
- [Schemas](./reference/schemas.md) - Well-known schemas and reserved schema operators.
