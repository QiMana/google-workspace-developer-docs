# Known issues

## Known issues

This document contains known issues for Google Cloud Search.

## API Explorer issues

The API Explorer might return OAuth errors due to user interface limitations. To avoid these errors, click **Show Scopes** under Google OAuth 2.0 and deselect all scopes except `https://www.googleapis.com/auth/cloud_search`.
