# Samples

This section provides links to the Google Cloud Search samples and source code on GitHub.

| Sample type | Description |
| --- | --- |
| [Reference connectors](https://github.com/google-cloudsearch) | The source code for six pre-built connectors and a Python content connector. Use this code to as reference to build your own connector. |
| [Connector SDK](https://github.com/google-cloudsearch/connector-sdk) | The source code for the connector SDK which is built upon the REST API. Examine this code to see how to build your own SDK or see how the SDK APIs map to the REST API. |
| [Content connector samples](https://github.com/googleworkspace/cloud-search-samples/tree/main/indexing/connector/sdk) | The full, graph, and list traversal samples used in the [Create a content connector](./guides/content-connector.md) guide. |
| [Identity connector sample](https://github.com/googleworkspace/cloud-search-samples/tree/main/identity/connector/sdk/full-sync) | The full sync identity connector sample used in the [Create an identity connector](./guides/identity-connector.md) guide. |
| [Get started guide sample](https://github.com/googleworkspace/cloud-search-samples/tree/main/end-to-end) | The getting started tutorial source code. |
