Automating Insights with Gemini & Google Apps Script by Ivan Kutil This post describes an automated system built with the YouTube Data API, Google Apps Script, and Vertex AI, that extracts and summarizes key information from YouTube videos. This system uses a Researcher agent to find relevant videos and save their details, and a Librarian agent which leverages Gemini AI on Vertex AI to watch the videos, summarize them, and compile those summaries into a Google Doc. This solution eliminates the need to manually watch and summarize videos, allowing you to focus on analyzing the information. Learn more Solutions Spotlight Time management: Make an agenda for meetings Automatically create agenda documents in Google Docs and attach them to your Google Calendar meetings. The script creates a document template for an agenda. When you update your calendar, the script checks to see if any events you own include "#agenda" in the description.
