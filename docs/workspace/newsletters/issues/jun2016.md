Request Invite API news Launched: Mobile Audit in the Reports API—The new feature audits activities on iOS and Android devices managed by Google Mobile Management, giving administrators the visibility into key activities occurring in their domain. Launched: Sheets API—The new Sheets API gives developers programmatic access to powerful features in the Sheets web and mobile interfaces, including charts and pivot tables. Learn more here. Announced: Slides API Preview—The new Slides API lets developers create and update presentations programmatically by exporting data from other tools into Slides. Learn more here. Launched: People API—Provides read-only access to authenticated users' contacts. Launched: Calendar Resources API—Enables Google for Work customers to manage their physical resources like conference rooms and printers by adding them to meetings.
