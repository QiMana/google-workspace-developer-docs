G Suite Developer Newsletter April 2017 Unleash the power of the Sheets API with Apps Script The new Advanced Sheets Service lets developers use all the richer features of the Sheets API via Apps Script. Learn more via a guest post by Google Developers Experts Bruce McPherson and Romain Vialard. Learn more How to create Quizzes in Google Forms with Apps Script Quizzes have been a popular way to give and collect feedback using Google Forms, but what if you could programmatically create, customize or even grade quizzes at scale? Our new Google Apps Script Forms Service lets you do that and more. Check it out API & Tooling news Launched: New Google Drive Metrics Now Accessible from Reports API—The new metrics methods provide greater visibility into how files are shared inside and outside of domain. Launched: Updates to End User Consent for 3rd-party Apps and Single Sign-on Providers—We are implementing a new redirect form to make it clear to users which account they are authenticating with, as well as the permissions they are granting to applications. Launched: A New Issue Tracker for G Suite Developers—The tracker allows for improved collaboration and transparency by enabling developers to see the issues that Google is actively working on. Announced: Gmail Add-ons Developer Preview —A new way for developers to bring the power of their app into Gmail with just one tap, across devices. Developer News AODocs and DocuSign announced an integration that allows organizations to further automate document signature workflow without leaving G Suite. Smartsheet announced an integration with Inbox by Gmail allowing users to take actions on notifications of updates from their sheets without leaving Inbox.
