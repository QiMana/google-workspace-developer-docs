The badge is displayed from the Marketplace homepage and the app details page. The badge also appears to Google Workspace administrators when they install or manage apps from the Google Admin console. Learn more AppSheet now included for more Google Workspace customers AppSheet is a no-code platform integrated with Google Workspace that allows anyone to quickly and easily build apps to automate workflows, improve productivity, and connect with their customers. With this update, almost all Google Workspace customers now have the AppSheet Core plan included. Find out more details and how to get started in the link below. Learn more Programmatically read and write working locations for a portion of the day with the Calendar API Recently, we introduced the ability for users to set working locations in Calendar that indicate where they’re working for specific portions of the day. Now, we’v e added the ability to programmatically read and write working locations for specific portions of the day. This update expands on the existing reading and writing functionality announced earlier this year. Learn more Create spaces, memberships, group chats, and more using the Google Chat API is now GA Last year, we announced that developers could use the Google Chat API to programmatically create new spaces and add members to those spaces through the Google Workspace Developer Preview Program. Now, those features are generally available for all Google Workspace developers with the Chat API.
