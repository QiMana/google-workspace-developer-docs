April, 2022Announcing the Apps Script connector for AppSheetThe Apps Script connector for AppSheet now makes it possible to call Apps Script code functions from a no-code AppSheet app. This greatly extends the abilities of AppSheet apps by letting them access the power that Apps Script provides.
