# 2025 april

April, 2025 Google Workspace developer announcements at Next’25 Google Cloud Next '25 focused heavily on AI integration and developer productivity for Google Workspace. Key takeaways for developers included the introduction of Google Workspace Flows, enabling AI-driven automation of multi-step workflows connecting apps like Forms, Sheets, and Chat. While AppSheet, the primary no-code platform, had a session focused on integrating existing AI capabilities. Apps Script was showcased in the Build Google Workspace add-ons with a boost from Gemini demo, showing how Gemini can be used to create code for solutions in Workspace. Finally, the event facilitated community engagement through dedicated meetups for various developer groups, including AppSheet developers, Apps Script developers, and Workspace admins. These meetups provided opportunities for networking and discussion. Additionally, the Developer Theater hosted lighting talks that explored specific technical applications, such as using Apps Script for AI finetuning and leveraging Gemini for AI and automation within Google Workspace.
