# Google Cloud Innovators and Google Workspace Newsletter

The Google Workspace Developer Newsletter is an emailed summary of the latest updates for Google Workspace developers, including new features, upcoming events, and more.

To subscribe, join the Google Cloud Innovators program and select "Google Workspace" as an interest within your [Developer Profile](https://developers.google.com/profile/u/me).
