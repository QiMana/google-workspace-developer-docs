# Google Tasks API release notes

To get the latest product updates delivered to you, add the URL of this page to your [feed reader](https://wikipedia.org/wiki/Comparison_of_feed_aggregators), or add the feed URL directly: `https://developers.google.com/feeds/tasks-release-notes.xml`.

This page contains release notes for features and updates to the Tasks API. We recommend that Tasks API developers periodically check this list for any new announcements.

## July 23, 2024

Feature

**Generally Available**: You can now get, edit, and delete tasks assigned from Google Docs documents or Chat spaces using the Tasks API. For more information, refer to the [tasks reference documentation](https://developers.google.com/tasks/reference/rest/v1/tasks).

## June 28, 2018

Announcement

The Google Tasks API is now generally available.
