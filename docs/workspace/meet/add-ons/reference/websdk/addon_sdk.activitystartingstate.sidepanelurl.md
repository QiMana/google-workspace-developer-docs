# Property signature ActivityStartingState.sidePanelUrl

## Property signature ActivityStartingState.sidePanelUrl

The URL that the side panel opens for users joining the activity. The length of this URL must be less than 512 characters.

## Signature

```
sidePanelUrl?: string;
```
