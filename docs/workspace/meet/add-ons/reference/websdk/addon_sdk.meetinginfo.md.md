# Interface MeetingInfo

Information about the meeting in which the add-on is running.

## Signature

```
interface MeetingInfo
```

## Property signatures

| Name | Description |
| --- | --- |
| [`meetingCode`](https://developers.google.com/workspace/meet/add-ons/reference/websdk/addon_sdk.meetinginfo.meetingcode.md) | The `meetingCode` for the current meeting in the form of aaa-bbbb-ccc. |
| [`meetingId`](https://developers.google.com/workspace/meet/add-ons/reference/websdk/addon_sdk.meetinginfo.meetingid.md) | The `meetingId` for the current meeting. Can be used to retrieve meeting information from the Meet REST API. The `meetingId` is a globally unique identifier for the meeting. |
