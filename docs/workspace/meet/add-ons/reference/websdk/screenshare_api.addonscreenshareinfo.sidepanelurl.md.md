# Property signature AddonScreenshareInfo.sidePanelUrl

## Property signature AddonScreenshareInfo.sidePanelUrl

The URL that the side panel opens after the add-on starts. Must belong to the same domain as the URLs specified in the add-on manifest.

## Signature

```
sidePanelUrl?: string;
```
