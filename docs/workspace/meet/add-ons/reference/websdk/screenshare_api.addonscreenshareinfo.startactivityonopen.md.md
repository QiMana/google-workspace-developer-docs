# Property signature AddonScreenshareInfo.startActivityOnOpen

## Property signature AddonScreenshareInfo.startActivityOnOpen

Whether to start an activity when the add-on is opened. If a mainstage URL is provided, must be true.

## Signature

```
startActivityOnOpen: boolean;
```

## Details

| Type | `boolean` |
| --- | --- |
| Optional | No |
| Readonly | No |
