# Property signature Participant.mediaEntries

## Property signature Participant.mediaEntries

The media entries associated with this participant. These can be transient. There is one participant to many media entries relationship.

## Signature

```
readonly mediaEntries: Subscribable<MediaEntry[]>;
```

## Details

| Type | `Subscribable<MediaEntry[]>` |
| --- | --- |
| Optional | No |
| Readonly | Yes |
