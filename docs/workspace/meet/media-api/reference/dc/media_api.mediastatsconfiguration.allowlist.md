# Property signature MediaStatsConfiguration.allowlist

## Property signature MediaStatsConfiguration.allowlist

A map of allow listed sections. The key is the section type, and the value is the keys that are allow listed for that section. Fields can be found in [RTCStatsReport](https://developer.mozilla.org/en-US/docs/Web/API/RTCStatsReport)

## Signature

```
allowlist: Map<string, {key: string[]}>;
```
