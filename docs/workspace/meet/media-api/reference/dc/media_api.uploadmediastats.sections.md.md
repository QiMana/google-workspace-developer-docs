# Property signature UploadMediaStats.sections

## Property signature UploadMediaStats.sections

Represents the entries in [RTCStatsReport](https://developer.mozilla.org/en-US/docs/Web/API/RTCStatsReport). Formatted as an array of objects with an id and a type. The value of the id is a string WebRTC id of the section. The value of the type is the section.

## Signature

```
sections: StatsSectionData[];
```

## Details

| Type | `StatsSectionData[]` |
| --- | --- |
| Optional | No |
| Readonly | No |
