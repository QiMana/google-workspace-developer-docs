# Interface CanvasAssignment

Video assignment for a single canvas.

## Signature

```
declare interface CanvasAssignment
```

## Property signatures

| Name | Description |
| --- | --- |
| [`canvasId`](https://developers.google.com/workspace/meet/media-api/reference/dc/media_api.canvasassignment.canvasid.md) | The video canvas the video should be shown in. |
| [`mediaEntryId`](https://developers.google.com/workspace/meet/media-api/reference/dc/media_api.canvasassignment.mediaentryid.md) | ID of the media entry associated with the video stream. |
| [`ssrc`](https://developers.google.com/workspace/meet/media-api/reference/dc/media_api.canvasassignment.ssrc.md) | The virtual video SSRC that the video will be sent over, or unset if no video from the participant. |
