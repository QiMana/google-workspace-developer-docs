# Google Docs API release notes

To get the latest product updates delivered to you, add the URL of this page to your [feed reader](https://wikipedia.org/wiki/Comparison_of_feed_aggregators), or add the feed URL directly: `https://developers.google.com/feeds/docs-release-notes.xml`.

This page contains release notes for features and updates to the Docs API. We recommend that Docs API developers periodically check this list for any new announcements.

## October 02, 2025

Deprecated

After October 14, 2025, calling [Gemini Code Assist `@googledocs` tool](https://developers.google.com/workspace/docs/api/how-tos/read-docs-gemini-tool) using the @ operator with the tool name won't work. Gemini Code Assist tools are replaced by [agent mode](https://developers.google.com/gemini-code-assist/docs/agent-mode).

## April 09, 2025

Announcement

[**Preview**](https://cloud.google.com/products#product-launch-stages): Use the `@googledocs` command to read, summarize, and list your Google Docs assets like documents and folders directly from your IDE by installing Gemini Code Assist. To learn more, see [Read documents while coding with Gemini Code Assist](https://developers.google.com/workspace/docs/api/how-tos/read-docs-gemini-tool).

## July 29, 2024

Feature

**Generally available**: You can now create and organize documents with tabs using the Google Docs API. For more information, see [Work with tabs](https://developers.google.com/docs/api/how-tos/tabs).

## February 04, 2019

Announcement

The Google Docs API is now generally available.
