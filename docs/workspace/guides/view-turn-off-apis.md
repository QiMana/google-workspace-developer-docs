# View & turn off Google Workspace APIs

To keep track of APIs and their usage, you can view a list of Google Workspace APIs that are turned on in your Cloud project. You can view detailed metrics about each API and turn off an API so that projects, client IDs, or service accounts can't make calls to an API.

To turn off an API, complete these steps:

1. In the Google Cloud console, click **Menu** \> the **View all products** tile \> **Google Workspace** \> **APIs**.
	[Go to APIs](https://console.cloud.google.com/workspace-api/api-list)
2. In the list of **Enabled APIs**, click the API that you want to turn off.
3. Click **Disable**.
4. In the **Disable API** box, click **Disable**.
