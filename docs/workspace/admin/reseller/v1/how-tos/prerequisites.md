# Prerequisites

The Reseller API is intended for authorized reseller developers and service integrators who want to write client applications that manage customer orders and subscriptions.

You must complete these steps before using the Reseller API:

1. Enroll as a Google Workspace authorized reseller. You must have a fully executed and signed reseller contract to use the Partner Sales Console and the Reseller API. For more information about becoming a reseller, contact the [Google Workspace reseller program](https://www.google.com/intl/en/enterprise/apps/business/landing/partners/resellers/).
2. Complete the steps to get started [developing on Google Workspace](../../../../guides/get-started.md).
3. Set up the [Partner Sales Console](https://support.google.com/channelservices/answer/9658468). The Reseller API is an interface for managing and creating subscriptions to the Partner Sales Console.
