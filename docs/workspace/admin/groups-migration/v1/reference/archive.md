# Archive

## Archive

For a list of [methods](#methods) for this resource, see the end of this page.

## Resource representations

The Groups Migration API archives shared emails into a group's archive using the media upload option.

## Methods

`insert`

Migrates an email message into the archive of the Google group using media upload.
