# Status

An outage's current status. Provides one of the options: New, Ongoing, Resolved, False Positive, Partially Resolved, and Merged.

<table><colgroup><col width="25%"> <col></colgroup><thead><tr><th colspan="2">Enums</th></tr></thead><tbody><tr><td><code>STATUS_UNSPECIFIED</code></td><td>Status is unspecified.</td></tr><tr><td><code>NEW</code></td><td>The incident has just been reported.</td></tr><tr><td><code>ONGOING</code></td><td>The incident is ongoing.</td></tr><tr><td><code>RESOLVED</code></td><td>The incident has been resolved.</td></tr><tr><td><code>FALSE_POSITIVE</code></td><td>Further assessment indicated no customer impact.</td></tr><tr><td><code>PARTIALLY_RESOLVED</code></td><td>The incident has been partially resolved.</td></tr><tr><td><code>MERGED</code></td><td>The incident was merged into a parent.</td></tr><tr><td><code>DOWNGRADED</code></td><td>The incident has lower impact than initially anticipated.</td></tr></tbody></table>
