# UserChanges

## UserChanges

Alerts from UserChanges bucket Rules for predefined rules which contain the following rules:

- Suspended user made active
- New user added
- User suspended (by admin)
- User granted admin privileges
- User admin privileges revoked
- User deleted
- Users password changed

JSON representation

```
{
  "name": string
}
```
