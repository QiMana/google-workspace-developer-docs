# Takeout Audit Activity Events

This document lists the events and parameters for various types of Takeout Audit activity events. You can retrieve these events by calling [Activities.list()](https://developers.google.com/workspace/admin/reports/v1/reference/activities/list) with `applicationName=takeout`.

## User Takeout

Events of this type are returned with `type=USER_TAKEOUT`.

### User completed a Takeout

An event that occurs when a user takeout request is completed.

Possible values:</p><ul><li><code>BOX</code><br>Box.</li><li><code>DRIVE</code><br>Google Drive.</li><li><code>DROPBOX</code><br>Dropbox.</li><li><code>EMAIL</code><br>Takeout delivered via email download link to local storage.</li><li><code>ONEDRIVE</code><br>Microsoft OneDrive.</li><li><code>UNKNOWN</code><br>Takeout delivery location is unknown.</li></ul></td></tr><tr><td><code>TAKEOUT_ID</code></td><td><p><code>string</code></p><p>Takeout job ID.</p></td></tr><tr><td><code>TAKEOUT_STATUS</code></td><td><p><code>string</code></p><p>Takeout status. Possible values:</p><ul><li><code>CANCELED</code><br>Takeout was canceled.</li><li><code>COMPLETED</code><br>Takeout was completed.</li><li><code>FAILED</code><br>Takeout failed.</li><li><code>IN_PROGRESS</code><br>Takeout is still in progress.</li></ul></td></tr><tr><td><code>USER_EMAIL</code></td><td><p><code>string</code></p><p>Email address of the target user.</p></td></tr></tbody></table></td></tr><tr><td>Sample request</td><td><div><code>GET https://admin.googleapis.com/admin/reports/v1/activity/users/all/applications/<b>takeout</b>?eventName=<b>COMPLETED_USER_TAKEOUT</b>&maxResults=10&access_token=<var>YOUR_ACCESS_TOKEN</var></code></div></td></tr><tr><td>Admin Console message format</td><td><div><code><var>{actor}</var> user takeout <var>{TAKEOUT_STATUS}</var></code></div></td></tr></tbody></table>
