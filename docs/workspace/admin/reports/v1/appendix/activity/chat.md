# Google Chat Audit Activity Events

This document lists the events and parameters for various types of Google Chat Audit activity events. You can retrieve these events by calling [Activities.list()](../../../reference/rest/v1/activities/list.md) with `applicationName=chat`.

## User Action

Events of this type are returned with `type=user_action`.
