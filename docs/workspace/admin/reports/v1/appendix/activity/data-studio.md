# Data Studio Audit Activity Events

This document lists the events and parameters for various types of Data Studio Audit activity events. You can retrieve these events by calling [Activities.list()](../../../reference/rest/v1/activities/list.md) with `applicationName=data_studio`.

## Access

Asset type related to asset accessing. Events of this type are returned with `type=ACCESS`.
