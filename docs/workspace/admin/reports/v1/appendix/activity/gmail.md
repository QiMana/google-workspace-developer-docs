# Gmail Activity Events

This document lists the events and parameters for various types of Gmail activity events. You can retrieve these events by calling [Activities.list()](../../../reference/rest/v1/activities/list.md) with `applicationName=gmail`.

## Gmail audit event type

The type of Gmail audit events exported to Google Workspace customers. Events of this type are returned with `type=delivery_type`.
