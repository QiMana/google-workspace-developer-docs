# Vault Audit Activity Events

This document lists the events and parameters for various types of Vault Audit activity events. You can retrieve these events by calling [Activities.list()](https://developers.google.com/workspace/admin/reports/v1/reference/activities/list) with `applicationName=vault`.

## Event Type: User Action

User Action Event Type. Events of this type are returned with `type=user_action`.
