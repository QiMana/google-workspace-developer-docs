# Admin Audit Activity Events - Drive Settings

This document lists the events and parameters for Drive Settings Admin Audit activity events. You can retrieve these events by calling [Activities.list()](https://developers.google.com/workspace/admin/reports/v1/reference/activities/list) with `applicationName=admin`.

## Drive Settings

Events of this type are returned with `type=DOCS_SETTINGS`.

### Document Ownership Change
