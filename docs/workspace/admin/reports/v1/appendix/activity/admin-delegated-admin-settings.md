# Admin Audit Activity Events - Delegated Admin Settings

This document lists the events and parameters for Delegated Admin Settings Admin Audit activity events. You can retrieve these events by calling [Activities.list()](https://developers.google.com/workspace/admin/reports/v1/reference/activities/list) with `applicationName=admin`.

## Delegated Admin Settings

Events of this type are returned with `type=DELEGATED_ADMIN_SETTINGS`.
