# Admin Audit Activity Events - Mobile Settings

This document lists the events and parameters for Mobile Settings Admin Audit activity events. You can retrieve these events by calling [Activities.list()](../../../reference/rest/v1/activities/list.md) with `applicationName=admin`.

## Mobile Settings

Events of this type are returned with `type=MOBILE_SETTINGS`.
