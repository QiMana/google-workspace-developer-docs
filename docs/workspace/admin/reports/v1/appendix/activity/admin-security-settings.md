# Admin Audit Activity Events - Security Settings

This document lists the events and parameters for Security Settings Admin Audit activity events. You can retrieve these events by calling [Activities.list()](https://developers.google.com/workspace/admin/reports/v1/reference/activities/list) with `applicationName=admin`.

## Security Settings

Events of this type are returned with `type=SECURITY_SETTINGS`.
