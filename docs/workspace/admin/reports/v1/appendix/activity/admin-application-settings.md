# Admin Audit Activity Events - Application Settings

This document lists the events and parameters for Application Settings Admin Audit activity events. You can retrieve these events by calling [Activities.list()](../../../reference/rest/v1/activities/list.md) with `applicationName=admin`.

## Application Settings

Events of this type are returned with `type=APPLICATION_SETTINGS`.
