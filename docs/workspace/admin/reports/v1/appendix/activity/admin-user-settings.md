# Admin Audit Activity Events - User Settings

This document lists the events and parameters for User Settings Admin Audit activity events. You can retrieve these events by calling [Activities.list()](../../../reference/rest/v1/activities/list.md) with `applicationName=admin`.

## User Settings

Events of this type are returned with `type=USER_SETTINGS`.
