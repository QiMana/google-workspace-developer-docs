# Classroom Audit Activity Events

This document lists the events and parameters for various types of Classroom Audit activity events. You can retrieve these events by calling [Activities.list()](https://developers.google.com/workspace/admin/reports/v1/reference/activities/list) with `applicationName=classroom`.

## Add-on updates

Name of a type of event including updates to Add-ons and their resources. Events of this type are returned with `type=add_on_update`.
