# Rules Audit Activity Events

This document lists the events and parameters for various types of Rules Audit activity events. You can retrieve these events by calling [Activities.list()](https://developers.google.com/workspace/admin/reports/v1/reference/activities/list) with `applicationName=rules`.

## Action complete type

Audit event type which indicates action complete events. Events of this type are returned with `type=action_complete_type`.
