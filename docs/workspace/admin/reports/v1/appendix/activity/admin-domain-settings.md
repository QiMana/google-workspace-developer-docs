# Admin Audit Activity Events - Domain Settings

This document lists the events and parameters for Domain Settings Admin Audit activity events. You can retrieve these events by calling [Activities.list()](../../../reference/rest/v1/activities/list.md) with `applicationName=admin`.

## Domain Settings

Events of this type are returned with `type=DOMAIN_SETTINGS`.
