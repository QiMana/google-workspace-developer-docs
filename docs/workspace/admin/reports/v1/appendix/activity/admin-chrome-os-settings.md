# Admin Audit Activity Events - ChromeOS Settings

This document lists the events and parameters for ChromeOS Settings Admin Audit activity events. You can retrieve these events by calling [Activities.list()](https://developers.google.com/workspace/admin/reports/v1/reference/activities/list) with `applicationName=admin`.

## ChromeOS Settings

Events of this type are returned with `type=CHROME_OS_SETTINGS`.
