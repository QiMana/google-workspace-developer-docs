# OAuth Token Audit Activity Events

This document lists the events and parameters for various types of OAuth Token Audit activity events. You can retrieve these events by calling [Activities.list()](https://developers.google.com/workspace/admin/reports/v1/reference/activities/list) with `applicationName=token`.

## Auth

Events of this type are returned with `type=auth`.

### Activity

Access activity for an application.
