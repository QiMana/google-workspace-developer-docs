# Google Meet Audit Activity Events

This document lists the events and parameters for various types of Google Meet Audit activity events. You can retrieve these events by calling [Activities.list()](../../../reference/rest/v1/activities/list.md) with `applicationName=meet`.

## Call event

A record describing a single meeting endpoint. Events of this type are returned with `type=call`.
