# Google Docs Organization Metrics (Deprecated)

| Google Drive and Google Docs (Deprecated) | Notes |
| --- | --- |
| num\_docs | Number of Docs (text documents, drawings, forms, presentations, spreadsheets, and uploaded files) within the customer's account that are not in Trash on the date of the report. |
| num\_docs\_edited | Number of Docs within the customer's account edited on the date of the report. |
| num\_docs\_not\_edited\_for\_3months | Number of Docs within the customer's account which have not been edited in the last 90 days. |
| num\_docs\_not\_edited\_for\_6months | Number of Docs within the customer's account which have not been edited in the last 180 days. |
| num\_docs\_not\_edited\_for\_12months | Number of Docs within the customer's account which have not been edited in the last 360 days. |
| num\_docs\_not\_viewed\_for\_3months | Number of Docs within the customer's account which have not been viewed in the last 90 days. |
| num\_docs\_not\_viewed\_for\_6months | Number of Docs within the customer's account which have not been viewed in the last 180 days. |
| num\_docs\_not\_viewed\_for\_12months | Number of Docs within the customer's account which have not been viewed in the last 360 days. |
| num\_docs\_shared\_outside\_domain | (DEPRECATED) Number of Docs within the customer's account that are not public or visible to anyone with a link, but explicitly are shared either with users or groups outside the domain up to the date of the report. |
| num\_docs\_viewed | Number of Docs within the customer's account viewed on the date of the report. |
| num\_docs\_with\_visibility\_anyone\_with\_link | Number of Docs within the customer's account which can be viewed by anyone having the link on the date of the report. |
| num\_docs\_with\_visibility\_people\_at\_domain | Number of Docs within the customer's account which can be viewed by anyone in the domain on the date of the report. |
| num\_docs\_with\_visibility\_people\_at\_domain\_with\_link | Number of Docs within the customer's account which can be viewed by anyone in the domain having the document link on the date of the report. |
| num\_docs\_with\_visibility\_private | Number of Docs within the customer's account which are either explicitly shared with users/groups or are private on the date of the report. |
| num\_docs\_with\_visibility\_public | Number of Docs within the customer's account which can be viewed by anyone on the web on the date of the report. |
| num\_docs\_externally\_visible | (DEPRECATED) Number of Docs in the customer's account visible to people outside the domain. This is the sum of `num_docs_with_visibility_public`, `num_docs_with_visibility_anyone_with_link`, and `num_docs_shared_outside_domain`. |
| num\_docs\_internally\_visible | (DEPRECATED) Number of Docs in the customer's account visible only to people inside the domain. This is the sum of `num_docs_with_visibility_people_at_domain`, `num_docs_with_visibility_people_at_domain_with_link`, and `num_docs_visibility_private`. |
| num\_drawings | Number of the customer account's drawings in [Docs format](http://developers.google.com/drive/integrate-open) that are not in Trash on the date of the report. |
| num\_drawings\_edited | Number of the customer account's drawings in [Docs format](http://developers.google.com/drive/integrate-open) edited on the date of the report. |
| num\_drawings\_viewed | Number of the customer account's drawings in [Docs format](http://developers.google.com/drive/integrate-open) viewed on the date of the report. |
| num\_forms | Number of forms within the customer's account that are not in Trash on the date of the report. |
| num\_forms\_edited | Number of forms within the customer's account edited on the date of the report. |
| num\_forms\_viewed | Number of forms within the customer's account viewed on the date of the report. |
| num\_presentations | Number of the customer account's presentations in [Docs format](http://developers.google.com/drive/integrate-open) that are not in Trash on the date of the report. |
| num\_presentations\_edited | Number of the customer account's presentations in [Docs format](http://developers.google.com/drive/integrate-open) edited on the date of the report. |
| num\_presentations\_viewed | Number of the customer account's presentations in [Docs format](http://developers.google.com/drive/integrate-open) viewed on the date of the report. |
| num\_shared\_docs | Number of Docs within the customer's account which are explicitly shared with users/groups, or are not private as on the date of the report. |
| num\_spreadsheets | Number of the customer account's spreadsheets in [Docs format](http://developers.google.com/drive/integrate-open) that are not in Trash on the date of the report. |
| num\_spreadsheets\_edited | Number of the customer account's spreadsheets in [Docs format](http://developers.google.com/drive/integrate-open) and edited on the date of the report. |
| num\_spreadsheets\_viewed | Number of the customer account's spreadsheets in [Docs format](http://developers.google.com/drive/integrate-open) viewed on the date of the report. |
| num\_text\_documents | Number of the customer account's text documents in [Docs format](http://developers.google.com/drive/integrate-open) that are not in Trash on the date of the report. |
| num\_text\_documents\_edited | Number of the customer account's text documents in [Docs format](http://developers.google.com/drive/integrate-open) edited on the date of the report. |
| num\_text\_documents\_viewed | Number of the customer account's text documents in [Docs format](http://developers.google.com/drive/integrate-open) viewed on the date of the report. |
| num\_uploaded\_files | Number of the customer account's uploaded files in non [Docs format](http://developers.google.com/drive/integrate-open) that are not in Trash on the date of the report. |
| num\_uploaded\_files\_edited | Number of the customer account's uploaded files in non [Docs format](http://developers.google.com/drive/integrate-open) edited on the date of the report. |
| num\_uploaded\_files\_viewed | Number of the customer account's uploaded files in non [Docs format](http://developers.google.com/drive/integrate-open) viewed on the date of the report. |
| num\_1day\_active\_users | Number of users who have used Docs on the day of the report |
| num\_7day\_active\_users | Number of users who have used Docs in the past 7 days from the date of the report |
| num\_30day\_active\_users | Number of users who have used Docs in the past 30 days from the date of the report |
